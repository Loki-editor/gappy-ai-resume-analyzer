# 🚀 Gappy AI Resume Analyzer

## Problem Statement

Fresh graduates often struggle to understand whether their resumes are ATS-friendly and suitable for specific job roles.

## Solution

Gappy AI Resume Analyzer is an AI-powered application that analyzes resumes, provides ATS scores, identifies missing skills, recommends career paths, and generates interview questions.

## Features

- Upload Resume (PDF)
- ATS Score
- Skill Gap Analysis
- AI Resume Suggestions
- Career Recommendations
- Interview Question Generator

## Tech Stack

- Python
- Flask
- HTML/CSS/JavaScript
- Gemma AI
- PyMuPDF

## Future Scope

- LinkedIn Profile Analysis
- Resume Builder
- AI Career Mentor
- Multi-language Support
